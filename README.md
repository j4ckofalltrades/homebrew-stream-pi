# homebrew-stream-pi

Unofficial Homebrew formulae for [Stream-Pi](https://stream-pi.com).

## Usage

Add Homebrew tap:

```sh
$ brew tap j4ckofalltrades/stream-pi
```

Install Stream-Pi client and/or server:

```sh
# client
$ brew install stream-pi-client

# server
$ brew install stream-pi-server

```
